# C-CareerHelperAppProject
C-CareerHelperAppProject is a focused C# application that guides users toward careers that match their skills and interests. Users take a structured test that measures abilities, preferences, and soft skills; the app scores responses, identifies strengths and weaknesses, and produces ranked profession suggestions with personalized learning recommendations and next steps.
Key features
- Interactive assessment engine that measures technical skills, soft skills, and interests.
- Strengths and weaknesses report with clear, actionable descriptions.
- Profession recommendations ranked by fit score and accompanied by role summaries.
- Personalized learning plan that suggests courses, skills to build, and small projects.
- Profile and progress tracking to save test results and track improvement over time.
- JSON persistence for easy export, import, and inspection of user data.
